# CS 135 Virtual Reality

## Best Demo Award Selection Rules

The Best Demo Award is given at the end of each quarter based on the final project demo sessions. The selection process combines judge evaluations and peer voting.

### Award Categories

1. **Best Demo**: Selected by both judge scores and people's vote, together ranked #1
2. **Judge's Favorite**: #1 from judge score (other than Best Demo)
3. **People's Choice**: #1 from people's vote (other than the above)
4. **Honorable Mention**: One from judge scores, one from people's vote (other than the above)

### Notes

- Each demo session votes independently; if the 2nd session has fewer voters, at least one honorable mention should be selected from that session to ensure fair representation.
- Judge scores are anonymized (labeled A/B/C/D) and averaged across all judges who evaluated each team.
- People's vote allows each student to vote for up to 3 teams within their session.
